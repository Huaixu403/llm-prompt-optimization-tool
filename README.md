# LLM-提示-优化工具（推广话术智能优化工具）

## 项目背景
针对线上推广话术转化率低、人工优化效率差的业务痛点，结合本人推广专员工作场景，基于大模型API开发轻量化话术优化工具，实现推广话术的智能生成、多版本迭代与效果评估，打通大模型应用与业务落地的衔接。

## 核心工作
- 基于Python完成通义千问/豆包大模型API对接与调用，开发话术批量生成、多版本对比核心功能
- 运用Prompt工程方法论，设计结构化提示词模板，优化话术精准度与场景适配度，有效降低大模型幻觉率
- 基于100组真实推广数据，完成话术效果对照测试，用Python Pandas完成数据清洗与统计分析，用Matplotlib绘制可视化对比图表
- 基于WSL2 Ubuntu环境开发调试，用Docker封装运行环境，实现跨设备一键部署使用

## 项目成果
- 话术优化效率提升60%，单条优质话术产出时间从30分钟缩短至5分钟
- 优化前客户触达转化率2%，优化后达到2.5%，相对提升25%，累计带来新增客户咨询120+
- 沉淀12套可复用推广场景Prompt模板，完成可演示Demo，适配不同行业、不同客群的推广话术生成需求

## 文件说明
- `llm_api_call.py`：大模型API调用核心逻辑伪代码，展示核心开发思路
- `prompt_templates/`：12套可复用Prompt模板（文本形式记录在test_report.md中）
- `test_report.md`：100组测试数据的统计分析报告
- `visualization_charts.md`：优化前后转化率对比可视化图表说明
